┌─────────────────────────────────────────────────────────────────────┐
│ Step 1: Open your project                                          │
│ Step 2: Find the expense in the list                               │

   - Cash

   - Bank Transfer

   - Other
✅ Payment Status ⭐ NEW
   - Paid (green)
   - Unpaid (red)
   - Partial (orange)

💡 USE CASES FOR QUICK EDIT
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Quick Payment Status Update
   ┌───────────────────────────────────────────────────────────────┐
   │ Scenario: Just paid a vendor                                 │
   │ Action: Click ✏️ → Change "Unpaid" to "Paid" → Save        │
   │ Time: 5 seconds!                                             │
   └───────────────────────────────────────────────────────────────┘

2. Fix Amount Quickly
   ┌───────────────────────────────────────────────────────────────┐
   │ Scenario: Typed wrong amount                                 │
   │ Action: Click ✏️ → Update amount → Save                     │
   │ Time: 3 seconds!                                             │
   └───────────────────────────────────────────────────────────────┘

3. Change Category
   ┌───────────────────────────────────────────────────────────────┐
   │ Scenario: Wrong category selected                            │
   │ Action: Click ✏️ → Select correct category → Save           │
   │ Time: 5 seconds!                                             │
   └───────────────────────────────────────────────────────────────┘

4. Update Payment Method
   ┌───────────────────────────────────────────────────────────────┐
   │ Scenario: Actually paid by UPI, not Cash                     │
   │ Action: Click ✏️ → Change to UPI → Save                     │
   │ Time: 4 seconds!                                             │
   └───────────────────────────────────────────────────────────────┘
